# DevOps Portfolio Website

A modern, responsive portfolio website built for a DevOps enthusiast and Computer Science undergraduate. This project showcases technical skills, projects, and educational background with a focus on clean design and performance.

## 🚀 Features

- **Responsive Design**: Optimized for all screen sizes (mobile, tablet, and desktop).
- **Interactive UI**: Smooth animations and transitions using custom CSS.
- **Project Showcase**: Detailed section for technical projects and contributions.
- **Skills & Certifications**: Visual representation of technical expertise and achievements.
- **Contact Integration**: Easy-to-use contact section for professional inquiries.

## 🛠️ Tech Stack

- **Frontend**: [React 19](https://react.dev/) + [TypeScript](https://www.typescriptlang.org/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Styling**: Custom Vanilla CSS (Modern CSS features)
- **Deployment**: Optimized for Vercel / Render / Cloudflare Pages

## 📦 Getting Started

### Prerequisites

- Node.js (v18.0.0 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

### Building for Production

To create an optimized production build:

```bash
npm run build
```

The output will be in the `dist` directory.
